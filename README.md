# TaskApi

API simples de tarefas criada com ASP.NET Core.

## Funcionalidades

- Listar tarefas
- Criar tarefas
- Atualizar tarefas
- Deletar tarefas

## Tecnologias utilizadas

- C#
- ASP.NET Core Web API
- .NET 8

## Como executar o projeto

1. Abra o projeto no terminal.
2. Execute o comando:

dotnet run

3. Acesse no navegador:

http://localhost:5203/swagger
